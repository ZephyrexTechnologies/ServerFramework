# Zephyrex Server Boilerplate
## Installation
### Cloning
```sh
git clone git@github.com:ZephyrexTechnologies/Server.git
```
### Requirements
- Python 3.10
```sh
pip install -r requirements.txt
python3 src/Server.py
```
### Environment
```
APP_NAME=
APP_URI=http://localhost:1109
```
